Mesquite — Wearable Motion Capture Platform

Arizona State University

Open-source, low-cost wearable motion-capture system using 15 ESP32-C3–powered IMU sensor pods on FreeRTOS to stream orientation data wirelessly. A Raspberry Pi Zero W hub synchronizes quaternions with a smartphone's WebXR SLAM world frame, rendering a real-time 3D skeleton in any WebSerial-compatible browser via Three.js.
