Autonomous Maze Navigation with 6DOF Robotic Arm

MyCobot Pro 600 · MATLAB + Python · Digital Twin

This project leverages a MyCobot Pro 600 robotic arm to autonomously solve and traverse a 4×4 rectangular maze by fusing digital-twin simulation with real-world execution, using camera-based perception and algorithmic path planning to compute and apply precise joint trajectories.

How It Works

The project demonstrates a fully integrated workflow that bridges digital simulation and real-world robotics:

  • Vision-Based Calibration: ArUco markers translate image pixels into physical coordinates for accurate spatial mapping.
  • BFS Path Planning: A breadth-first search algorithm determines the optimal sequence of waypoints through the maze.
  • Inverse Kinematics: MATLAB's IK solver, called via a Python interface, generates precise joint trajectories for each waypoint.
  • Digital Twin Validation: All motions are verified in simulation before execution on the real hardware.

Results

The close alignment between the digital twin's simulated motions and the robot's actual performance validates the accuracy of the calibration and control pipeline. This work underscores the effectiveness of combining computer vision, algorithmic planning, and kinematic modeling for autonomous robotic tasks, and provides a scalable foundation for more complex navigation challenges.
